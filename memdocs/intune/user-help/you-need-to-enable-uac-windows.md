# <a name="how-to-enable-user-access-control"></a>如何启用用户访问控制

你的单位或学校希望确保访问其资源的设备受到保护。 要确保将设备设置为具有对这些资源的访问权限，方法之一是避免在设备上安装未经授权的软件。

下面介绍了如何更改用户访问控制的设置：

1. 转到“开始”并打开“控制面板”   。
2. 打开“安全和维护” **“更改用户帐户控制设置”**  >   。
3. 在“选择何时通知你有关计算机更改的消息”页中，将滑块移动至最接近“始终通知”的头两个上凹口之一   。
