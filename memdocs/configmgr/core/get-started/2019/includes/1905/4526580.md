## <a name="view-smbios-guid-in-device-lists"></a><a name="bkmk_smbios"></a> 查看设备列表中的 SMBIOS GUID

<!--4526580-->

在设备和设备集合节点中，现在可以为“SMBIOS GUID”添加新列  。 此值与 System Resource 类的“BIOS GUID”属性相同  。 它是设备硬件的唯一标识符。
